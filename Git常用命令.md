##git命令
---
参考文献：
http://www.cnblogs.com/springbarley/archive/2012/11/03/2752984.html

**放弃修改**

***1 tracked文件的撤销***

git checkout -- + filename

***2 unchecked文件的撤销***

git clean -df

***3 给当前版本打标签***

**显示所有的tags：**

git tag 

**给当前分支当前版本打tag：**

git tag + v版本号 

**打上附带说明的tag：**

git tag -a + v版本号 -m "说明"  

**显示tag及详细说明：**

git show + v版本号  

**删除tag：**

git -d + v版本号
